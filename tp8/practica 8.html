Ejercicios de Matemáticas

Bienvenidos a esta página de ejercicios de matemáticas para practicar y mejorar tus habilidades.

A continuación, encontrarás dos ejercicios para resolver:

Ejercicio 1

Resuelve la siguiente ecuación cuadrática para x:

2x^2 - 5x + 2 = 0

Recuerda que puedes utilizar la fórmula cuadrática para encontrar las soluciones.

Ejercicio 2

Calcula el valor de sen(π/4) utilizando la identidad trigonométrica fundamental:

sen^2(x) + cos^2(x) = 1

Luego, determina el valor de cos(π/4) utilizando la relación entre sen(x) y cos(x).
